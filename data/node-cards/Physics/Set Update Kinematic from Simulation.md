## Set Update Kinematic from Simulation
**Category:** Physics
**Search:** "Set Update Kinematic from Simulation"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- Update Kinematic from Simulation (boolean)

**Outputs:**
- Out (exec)

**Description:** Enables/disables whether this component should be updated by simulation when it is kinematic. This is needed if (for example) its velocity needs to be accessed.

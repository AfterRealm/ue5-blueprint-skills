## Set PhysicalMaterial Override
**Category:** Physics
**Search:** "Set PhysicalMaterial Override"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- New Phys Material (object)

**Outputs:**
- Out (exec)

**Description:** Changes the current PhysMaterialOverride for this component. Note that if physics is already running on this component, this will not alter its mass/inertia etc, it will only change its surface properties like friction.

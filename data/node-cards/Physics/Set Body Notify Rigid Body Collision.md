## Set Body Notify Rigid Body Collision
**Category:** Physics
**Search:** "Set Body Notify Rigid Body Collision"
**Target:** Skeletal Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- New Notify Rigid Body Collision (boolean) — The value to assign to bNotifyRigidBodyCollision
- Bone Name (name) — Name of the body to turn hit notifies on/off. None implies root body

**Outputs:**
- Out (exec)

**Description:** Changes the value of bNotifyRigidBodyCollision for a given body

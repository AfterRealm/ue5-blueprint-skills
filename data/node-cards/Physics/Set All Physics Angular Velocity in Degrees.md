## Set All Physics Angular Velocity in Degrees
**Category:** Physics
**Search:** "Set All Physics Angular Velocity in Degrees"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- New Ang Vel (vector) — New angular velocity to apply to physics, in degrees per second.
- Add to Current (boolean) — If true, NewAngVel is added to the existing angular velocity of all bodies.

**Outputs:**
- Out (exec)

**Description:** Set the angular velocity of all bodies in this component.

## Set Enable Gravity
**Category:** Physics
**Search:** "Set Enable Gravity"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- Gravity Enabled (boolean)

**Outputs:**
- Out (exec)

**Description:** Enables/disables whether this component is affected by gravity. This applies only to components with bSimulatePhysics set to true.

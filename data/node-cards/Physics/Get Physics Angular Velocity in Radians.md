## Get Physics Angular Velocity in Radians
**Category:** Physics
**Search:** "Get Physics Angular Velocity in Radians"
**Target:** Primitive Component

**Inputs:**
- Target (object)
- Bone Name (name) — If a SkeletalMeshComponent, name of body to get velocity of. 'None' indicates root body.

**Outputs:**
- Return Value (vector)

**Description:** Get the angular velocity of a single body, in radians per second.

## Get Physics Linear Velocity at Point
**Category:** Physics
**Search:** "Get Physics Linear Velocity at Point"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- Point (vector) — Point is specified in world space.
- Bone Name (name) — If a SkeletalMeshComponent, name of body to get velocity of. 'None' indicates root body.

**Outputs:**
- Out (exec)
- Return Value (vector)

**Description:** Get the linear velocity of a point on a single body.

## Set Angular Limits
**Category:** Physics
**Search:** "Set Angular Limits"
**Target:** Skeletal Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- In Bone Name (name) — Name of bone to adjust constraint ranges for
- Swing 1Limit Angle (real) — Size of limit in degrees, 0 means locked, 180 means free
- Twist Limit Angle (real) — Size of limit in degrees, 0 means locked, 180 means free
- Swing 2Limit Angle (real) — Size of limit in degrees, 0 means locked, 180 means free

**Outputs:**
- Out (exec)

**Description:** Sets the Angular Motion Ranges for a named constraint

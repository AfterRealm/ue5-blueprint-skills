## Set Physics Max Angular Velocity in Radians
**Category:** Physics
**Search:** "Set Physics Max Angular Velocity in Radians"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- New Max Ang Vel (real) — New maximum angular velocity to apply to body, in radians per second.
- Add to Current (boolean) — If true, NewMaxAngVel is added to the existing maximum angular velocity of the body.
- Bone Name (name) — If a SkeletalMeshComponent, name of body to modify maximum angular velocity of. 'None' indicates root body.

**Outputs:**
- Out (exec)

**Description:** Set the maximum angular velocity of a single body.

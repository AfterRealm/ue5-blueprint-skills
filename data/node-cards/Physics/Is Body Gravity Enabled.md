## Is Body Gravity Enabled
**Category:** Physics
**Search:** "Is Body Gravity Enabled"
**Target:** Skeletal Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- Bone Name (name) — The name of the bone to check.

**Outputs:**
- Out (exec)
- Return Value (boolean) — True if gravity is enabled on the bone.

**Description:** Checks whether or not gravity is enabled on the given bone. NAME_None indicates the root body should be queried. If the bone name given is otherwise invalid, false is returned.

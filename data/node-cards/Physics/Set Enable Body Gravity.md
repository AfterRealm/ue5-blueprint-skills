## Set Enable Body Gravity
**Category:** Physics
**Search:** "Set Enable Body Gravity"
**Target:** Skeletal Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- Enable Gravity (boolean) — Whether gravity should be enabled or disabled.
- Bone Name (name) — The name of the bone to modify.

**Outputs:**
- Out (exec)

**Description:** Enables or disables gravity for the given bone. NAME_None indicates the root body will be edited. If the bone name given is otherwise invalid, nothing happens.

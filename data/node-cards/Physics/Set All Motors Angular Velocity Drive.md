## Set All Motors Angular Velocity Drive
**Category:** Physics
**Search:** "Set All Motors Angular Velocity Drive"
**Target:** Skeletal Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- Enable Swing Drive (boolean)
- Enable Twist Drive (boolean)
- Skip Custom Physics Type (boolean)

**Outputs:**
- Out (exec)

**Description:** Enable or Disable AngularVelocityDrive. If motor is in SLERP mode it will be turned on if either EnableSwingDrive OR EnableTwistDrive are enabled. In Twist and Swing mode the twist and the swing can be controlled individually.

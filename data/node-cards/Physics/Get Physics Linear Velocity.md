## Get Physics Linear Velocity
**Category:** Physics
**Search:** "Get Physics Linear Velocity"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- Bone Name (name) — If a SkeletalMeshComponent, name of body to get velocity of. 'None' indicates root body.

**Outputs:**
- Out (exec)
- Return Value (vector)

**Description:** Get the linear velocity of a single body.

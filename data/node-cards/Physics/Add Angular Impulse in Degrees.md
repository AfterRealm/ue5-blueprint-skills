## Add Angular Impulse in Degrees
**Category:** Physics
**Search:** "Add Angular Impulse in Degrees"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- Impulse (vector)
- Bone Name (name) — If a SkeletalMeshComponent, name of body to apply angular impulse to. 'None' indicates root body.
- Vel Change (boolean) — If true, the Strength is taken as a change in angular velocity instead of an impulse (ie. mass will have no effect).

**Outputs:**
- Out (exec)

**Description:** Add an angular impulse to a single rigid body. Good for one time instant burst.

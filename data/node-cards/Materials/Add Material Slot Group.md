## Add Material Slot Group
**Category:** Materials
**Search:** "Add Material Slot Group"
**Target:** Nanite Assembly Builder

**Inputs:**
- In (exec)
- Target (object)

**Outputs:**
- Out (exec)
- Return Value (integer)

**Description:** Creates a new material slot group on the builder, which allows you to be selective about which assembly parts' materials can be merged together. Use the return value when adding a part to the assembly.

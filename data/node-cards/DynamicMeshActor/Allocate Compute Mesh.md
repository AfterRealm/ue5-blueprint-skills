## Allocate Compute Mesh
**Category:** DynamicMeshActor
**Search:** "Allocate Compute Mesh"
**Target:** Dynamic Mesh Actor

**Inputs:**
- In (exec)
- Target (object)

**Outputs:**
- Out (exec)
- Return Value (object)

**Description:** Request a compute mesh from the Pool, which will return a previously-allocated mesh or add and return a new one. If the Pool is disabled, a new UDynamicMesh will be allocated and returned.

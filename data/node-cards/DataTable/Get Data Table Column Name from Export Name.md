## Get Data Table Column Name from Export Name
**Category:** DataTable
**Search:** "Get Data Table Column Name from Export Name"
**Target:** Data Table Function Library

**Inputs:**
- In (exec)
- Table (object)
- Column Export Name (string)

**Outputs:**
- Out (exec)
- Out Column Name (name)
- Return Value (boolean) — True if a column was found for the friendly name, false otherwise.

**Description:** Get the raw property name of a data table column from its friendly export name.

## Get Data Table Column as String
**Category:** DataTable
**Search:** "Get Data Table Column as String"
**Target:** Data Table Function Library

**Inputs:**
- In (exec)
- Data Table (object)
- Property Name (name)

**Outputs:**
- Out (exec)
- Return Value (string)

**Description:** Export from the DataTable all the row for one column. Export it as string. The row name is not included. See: GetDataTableColumnNames. See: GetDataTableColumnNameFromExportName.

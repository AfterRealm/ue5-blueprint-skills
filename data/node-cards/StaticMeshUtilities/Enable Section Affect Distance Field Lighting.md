## Enable Section Affect Distance Field Lighting
**Category:** StaticMeshUtilities
**Search:** "Enable Section Affect Distance Field Lighting"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to enable/disable section affecting distance field lighting from.
- Affect Distance Field Lighting (boolean) — If the section affects distance field lighting.
- LODIndex (integer) — Index of the StaticMesh LOD.
- Section Index (integer) — Index of the StaticMesh Section.

**Outputs:**
- Out (exec)

**Description:** Enables/disables mesh section affects distance field lighting for a specific LOD.

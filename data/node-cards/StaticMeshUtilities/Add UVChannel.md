## Add UVChannel
**Category:** StaticMeshUtilities
**Search:** "Add UVChannel"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh on which to add a UV channel.
- LODIndex (integer) — Index of the StaticMesh LOD.

**Outputs:**
- Out (exec)
- Return Value (boolean) — true if a UV channel was added.

**Description:** Adds an empty UV channel at the end of the existing channels on the given LOD of a StaticMesh.

## Enable Section Cast Shadow
**Category:** StaticMeshUtilities
**Search:** "Enable Section Cast Shadow"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to enable/disable section's shadow casting from.
- Cast Shadow (boolean) — If the section should cast shadow.
- LODIndex (integer) — Index of the StaticMesh LOD.
- Section Index (integer) — Index of the StaticMesh Section.

**Outputs:**
- Out (exec)

**Description:** Enables/disables mesh section shadow casting for a specific LOD.

## Get Lod Count
**Category:** StaticMeshUtilities
**Search:** "Get Lod Count"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Mesh to process.

**Outputs:**
- Out (exec)
- Return Value (integer) — the number of LODs present on the input mesh. An negative value indicates that the command could not be executed. See log for explanation.

**Description:** Get number of LODs present on a static mesh.

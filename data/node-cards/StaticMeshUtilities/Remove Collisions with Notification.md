## Remove Collisions with Notification
**Category:** StaticMeshUtilities
**Search:** "Remove Collisions with Notification"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to remove collisions from.
- Apply Changes (boolean) — Indicates if changes must be apply or not.

**Outputs:**
- Out (exec)
- Return Value (boolean) — A boolean indicating if the removal was successful or not.

**Description:** Remove collisions from a static mesh. This method replicates what is done when invoking menu entries "Collision > Remove Collision" in the Mesh Editor.

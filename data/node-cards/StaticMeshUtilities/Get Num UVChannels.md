## Get Num UVChannels
**Category:** StaticMeshUtilities
**Search:** "Get Num UVChannels"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to query.
- LODIndex (integer) — Index of the StaticMesh LOD.

**Outputs:**
- Out (exec)
- Return Value (integer) — the number of UV channels.

**Description:** Returns the number of UV channels for the given LOD of a StaticMesh.

## Remove UVChannel
**Category:** StaticMeshUtilities
**Search:** "Remove UVChannel"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh on which to remove the UV channel.
- LODIndex (integer) — Index of the StaticMesh LOD.
- UVChannel Index (integer) — Index where to remove the UV channel.

**Outputs:**
- Out (exec)
- Return Value (boolean) — true if the UV channel was removed.

**Description:** Removes the UV channel at the specified channel index on the given LOD of a StaticMesh.

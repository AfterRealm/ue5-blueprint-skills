## Is Section Cast Shadow Enabled
**Category:** StaticMeshUtilities
**Search:** "Is Section Cast Shadow Enabled"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to check if its section is casting shadows.
- LODIndex (integer) — Index of the StaticMesh LOD.
- Section Index (integer) — Index of the StaticMesh Section.

**Outputs:**
- Out (exec)
- Return Value (boolean)

**Description:** Checks if a specific LOD mesh section can cast shadows.

## Enable Section Force Opaque
**Category:** StaticMeshUtilities
**Search:** "Enable Section Force Opaque"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to make section forcefully opaque from.
- Force Opaque (boolean) — If the section is forced to be opaque.
- LODIndex (integer) — Index of the StaticMesh LOD.
- Section Index (integer) — Index of the StaticMesh Section.

**Outputs:**
- Out (exec)

**Description:** Set/unset mesh section to be forcefully opaque for a specific LOD.

## Enable Section Visible in Ray Tracing
**Category:** StaticMeshUtilities
**Search:** "Enable Section Visible in Ray Tracing"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Static Mesh (object) — Static mesh to enable/disable section's visibility in ray tracing from.
- Visible in Ray Tracing (boolean) — If the section visible in ray tracing.
- LODIndex (integer) — Index of the StaticMesh LOD.
- Section Index (integer) — Index of the StaticMesh Section.

**Outputs:**
- Out (exec)

**Description:** Enables/disables mesh section's visibility in ray tracing for a specific LOD.

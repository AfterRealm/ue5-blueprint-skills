## Is Section Collision Enabled
**Category:** StaticMeshUtilities
**Search:** "Is Section Collision Enabled"
**Target:** Static Mesh Editor Subsystem

**Inputs:**
- Target (object)
- Static Mesh (object) — Static mesh to check section collisions from.
- LODIndex (integer) — Index of the StaticMesh LOD.
- Section Index (integer) — Index of the StaticMesh Section.

**Outputs:**
- Return Value (boolean) — True if the collision is enabled for the specified LOD of the StaticMesh section.

**Description:** Checks if a specific LOD mesh section has collision.

## Get Data Table Row
**Category:** Utilities
**Search:** "Get Data Table Row"

**Inputs:**
- In (exec)
- Data Table (object) — Data Table Object Reference Data TableThe DataTable you want to retreive a row from
- Row Name (name) — Name Row NameThe name of the row to retrieve from the DataTable

**Outputs:**
- Row Found (exec)
- Row Not Found (exec)
- Out Row (wildcard) — Wildcard Out RowThe returned TableRow, if found

**Description:** Attempts to retrieve a TableRow from a DataTable via it's RowName

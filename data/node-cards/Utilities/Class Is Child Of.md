## Class Is Child Of
**Category:** Utilities
**Search:** "Class Is Child Of"
**Target:** Kismet Math Library

**Inputs:**
- Test Class (class)
- Parent Class (class)

**Outputs:**
- Return Value (boolean) — true if TestClass == ParentClass, or if TestClass is a child of ParentClass; false otherwise, or if either the value for either parameter is 'None'.

**Description:** Determine if a class is a child of another class.

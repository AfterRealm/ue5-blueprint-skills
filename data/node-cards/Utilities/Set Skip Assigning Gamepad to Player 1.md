**Category:** Utilities
**Search:** "Set Skip Assigning Gamepad to Player 1"
**Target:** Game Maps Settings

**Inputs:**
- In (exec)
- Target (object)
- Skip First Player (boolean) — If set connected game pads will only be assigned to the second and subsequent players

**Outputs:**
- Out (exec)

**Description:** Modify "Skip Assigning Gamepad to Player 1" GameMapsSettings option

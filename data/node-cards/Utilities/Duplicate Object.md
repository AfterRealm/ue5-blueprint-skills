## Duplicate Object
**Category:** Utilities
**Search:** "Duplicate Object"
**Target:** Kismet System Library

**Inputs:**
- In (exec)
- Source Object (object) — The object being duplicated
- Outer (object) — The outer to use for the new instance
- Name (name) — The optional name of the new instance

**Outputs:**
- Out (exec)
- Return Value (object) — The duplicated object or null if it failed for some reason

**Description:** Duplicate an object to a new instance

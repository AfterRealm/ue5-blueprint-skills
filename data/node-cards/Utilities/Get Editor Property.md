## Get Editor Property
**Category:** Utilities
**Search:** "Get Editor Property"
**Target:** Kismet System Library

**Inputs:**
- In (exec)
- Object (object) — The object you want to retrieve a property value from.
- Property Name (name) — The name of the object property to retrieve the value from.

**Outputs:**
- Out (exec)
- Property Value (wildcard) — The retrieved property value, if found.
- Success? (boolean) — Whether the property value was found and correctly retrieved.

**Description:** Attempts to retrieve the value of a named property from the given object.

## Make Brush From Soft Texture
**Category:** Utilities
**Search:** "Make Brush From Soft Texture"

**Inputs:**
- In (exec)
- Texture (softobject)
- Width (integer)
- Height (integer)

**Outputs:**
- Out (exec)
- Completed (exec)
- Slate Brush (struct)

**Description:** Asynchronously loads a Soft Texture Reference and returns a slate brush using that texture on successful load

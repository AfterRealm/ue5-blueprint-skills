## Make Brush From Soft Material
**Category:** Utilities
**Search:** "Make Brush From Soft Material"

**Inputs:**
- In (exec)
- Material (softobject)
- Width (integer)
- Height (integer)

**Outputs:**
- Out (exec)
- Completed (exec)
- Slate Brush (struct)

**Description:** Asynchronously loads a Soft Material Reference and returns a slate brush using that material on successful load

## Get Horizontal Dot Product To
**Category:** Transformation
**Search:** "Get Horizontal Dot Product To"
**Target:** Actor

**Inputs:**
- Target (object)
- Other Actor (object)

**Outputs:**
- Return Value (real)

**Description:** Returns the dot product from this Actor to OtherActor, ignoring Z. Returns -2.0 on failure. Returns 0.0 for coincidental actors.

## Get Bone Transform
**Category:** Transformation
**Search:** "Get Bone Transform"
**Target:** Skinned Mesh Component

**Inputs:**
- Target (object)
- In Bone Name (name) — Name of the the bone to get the transform
- Transform Space (enum)

**Outputs:**
- Return Value (transform) — Bone transform in world space if bone is found. Otherwise it will return component's transform in world space.

**Description:** Get world-space bone transform.

## Detach From Actor
**Category:** Transformation
**Search:** "Detach From Actor"
**Target:** Actor

**Inputs:**
- In (exec)
- Target (object)
- Location Rule (enum) — How to handle translation when detaching.
- Rotation Rule (enum) — How to handle rotation when detaching.
- Scale Rule (enum) — How to handle scale when detaching.

**Outputs:**
- Out (exec)

**Description:** Detaches the RootComponent of this Actor from any SceneComponent it is currently attached to.

## Get Socket Transform
**Category:** Transformation
**Search:** "Get Socket Transform"
**Target:** Scene Component

**Inputs:**
- Target (object)
- In Socket Name (name) — Name of the socket or the bone to get the transform
- Transform Space (enum)

**Outputs:**
- Return Value (transform) — Socket transform in world space if socket if found. Otherwise it will return component's transform in world space.

**Description:** Get world-space socket transform.

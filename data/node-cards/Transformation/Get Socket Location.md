## Get Socket Location
**Category:** Transformation
**Search:** "Get Socket Location"
**Target:** Scene Component

**Inputs:**
- Target (object)
- In Socket Name (name) — Name of the socket or the bone to get the transform

**Outputs:**
- Return Value (vector) — Socket transform in world space if socket is found. Otherwise it will return component's transform in world space.

**Description:** Get world-space socket or bone location.

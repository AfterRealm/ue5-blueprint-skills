## Get Children Components
**Category:** Components
**Search:** "Get Children Components"
**Target:** Scene Component

**Inputs:**
- Target (object)
- Include All Descendants (boolean) — Whether to include all descendants in the list of children (i.e. grandchildren, great grandchildren, etc.)

**Outputs:**
- Children (object) — The list of attached child components

**Description:** Gets all components that are attached to this component, possibly recursively

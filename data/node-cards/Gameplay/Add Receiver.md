## Add Receiver
**Category:** Gameplay
**Search:** "Add Receiver"
**Target:** Game Framework Component Manager

**Inputs:**
- In (exec)
- Target (object)
- Receiver (object)
- Add Only in Game Worlds (boolean)

**Outputs:**
- Out (exec)

**Description:** Adds an actor as a receiver for components. If it passes the actorclass filter on requests it will get the components.

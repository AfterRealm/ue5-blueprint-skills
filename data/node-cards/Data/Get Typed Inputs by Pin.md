## Get Typed Inputs by Pin
**Category:** Data
**Search:** "Get Typed Inputs by Pin"
**Target:** PCGData Function Library

**Inputs:**
- In (exec)
- In Collection (struct)
- In Pin (struct)
- In Data Type Class (class)

**Outputs:**
- Out (exec)
- Out Tagged Data (struct)
- Return Value (object)

**Description:** Gets all inputs of the given class type and on the given pin, returning matching tagged data in the OutTaggedData value too

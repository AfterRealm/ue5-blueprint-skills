## Add to Collection
**Category:** Data
**Search:** "Add to Collection"
**Target:** PCGData Function Library

**Inputs:**
- In (exec)
- In Collection (struct)
- In Data (object)
- In Pin Label (name)
- In Tags (string)

**Outputs:**
- Out (exec)

**Description:** Adds a data object to a given collection, simpler usage than making a PCGTaggedData object. InTags can be empty.

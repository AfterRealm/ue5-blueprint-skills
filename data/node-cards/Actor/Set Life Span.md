## Set Life Span
**Category:** Actor
**Search:** "Set Life Span"
**Target:** Actor

**Inputs:**
- In (exec)
- Target (object)
- In Lifespan (real)

**Outputs:**
- Out (exec)

**Description:** Set the lifespan of this actor. When it expires the object will be destroyed. If requested lifespan is 0, the timer is cleared and the actor will not be destroyed.

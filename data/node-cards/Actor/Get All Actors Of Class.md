## Get All Actors Of Class
**Category:** Actor
**Search:** "Get All Actors Of Class"
**Target:** Gameplay Statics

**Inputs:**
- In (exec)
- Actor Class (class) — Class of Actor to find. Must be specified or result array will be empty.

**Outputs:**
- Out (exec)
- Out Actors (object) — Output array of Actors of the specified class.

**Description:** Find all Actors in the world of the specified class. This will be slow if there are many actors of the specified class.

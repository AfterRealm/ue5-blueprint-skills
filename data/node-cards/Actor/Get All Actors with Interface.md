## Get All Actors with Interface
**Category:** Actor
**Search:** "Get All Actors with Interface"
**Target:** Gameplay Statics

**Inputs:**
- In (exec)
- Interface (class) — Interface to find. Must be specified or result array will be empty.

**Outputs:**
- Out (exec)
- Out Actors (object) — Output array of Actors of the specified interface.

**Description:** Find all Actors in the world with the specified interface. This is a very slow operation, as it will search over every actor in the world.

## Set Actor as Active
**Category:** Actor
**Search:** "Set Actor as Active"
**Target:** AIESelection Sets

**Inputs:**
- In (exec)
- Target (object)
- In Actor (object)
- Set Active (boolean)

**Outputs:**
- Out (exec)
- Return Value (boolean)

**Description:** Set the AActor as Active or not, so it will or won't be selected from a selection set. Will return false if it doesn't match any sets*. Note if GetShowAndSetSelectedOnly() is true the selected state of the Actor will take precedence over this

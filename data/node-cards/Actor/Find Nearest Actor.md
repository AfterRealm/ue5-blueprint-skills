## Find Nearest Actor
**Category:** Actor
**Search:** "Find Nearest Actor"
**Target:** Gameplay Statics

**Inputs:**
- Origin (vector) — World Location from which the distance is measured.
- Actors to Check (object) — Array of Actors to examine and return Actor nearest to Origin.

**Outputs:**
- Distance (real) — Distance from Origin to the returned Actor.
- Return Value (object) — Nearest Actor.

**Description:** Returns an Actor nearest to Origin from ActorsToCheck array.

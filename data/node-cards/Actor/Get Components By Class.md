## Get Components By Class
**Category:** Actor
**Search:** "Get Components By Class"
**Target:** Actor

**Inputs:**
- Target (object)
- Component Class (class)

**Outputs:**
- Return Value (object)

**Description:** Gets all the components that inherit from the given class. Currently returns an array of UActorComponent which must be cast to the correct type. This intended to only be used by blueprints. Use GetComponents() in C++.

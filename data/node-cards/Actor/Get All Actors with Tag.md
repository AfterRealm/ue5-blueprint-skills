## Get All Actors with Tag
**Category:** Actor
**Search:** "Get All Actors with Tag"
**Target:** Gameplay Statics

**Inputs:**
- In (exec)
- Tag (name) — Tag to find. Must be specified or result array will be empty.

**Outputs:**
- Out (exec)
- Out Actors (object) — Output array of Actors of the specified tag.

**Description:** Find all Actors in the world with the specified tag. This is a very slow operation, as it will search over every actor in the world.

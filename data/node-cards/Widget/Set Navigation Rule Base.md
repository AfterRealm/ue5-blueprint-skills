## Set Navigation Rule Base
**Category:** Widget
**Search:** "Set Navigation Rule Base"
**Target:** Widget

**Inputs:**
- In (exec)
- Target (object)
- Direction (enum)
- Rule (enum) — The rule to use when navigation is taking place

**Outputs:**
- Out (exec)

**Description:** Sets the widget navigation rules for a specific direction. This can only be called on widgets that are in a widget tree. This works only for non Explicit, non Custom and non CustomBoundary Rules.

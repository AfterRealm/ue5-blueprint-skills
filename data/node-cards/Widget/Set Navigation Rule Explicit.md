## Set Navigation Rule Explicit
**Category:** Widget
**Search:** "Set Navigation Rule Explicit"
**Target:** Widget

**Inputs:**
- In (exec)
- Target (object)
- Direction (enum)
- In Widget (object) — Focus on this widget instance

**Outputs:**
- Out (exec)

**Description:** Sets the widget navigation rules for a specific direction. This can only be called on widgets that are in a widget tree. This works only for Explicit Rule.

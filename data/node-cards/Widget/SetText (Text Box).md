## SetText (Text Box)
**Category:** Widget
**Search:** "SetText (Text Box)"
**Target:** Text Box

**Inputs:**
- In (exec)
- Target (object)
- In Text (text) — The text to assign to the widget

**Outputs:**
- Out (exec)

**Description:** Directly sets the widget text. Warning: This will wipe any binding created for the Text property!

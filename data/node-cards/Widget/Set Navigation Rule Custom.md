## Set Navigation Rule Custom
**Category:** Widget
**Search:** "Set Navigation Rule Custom"
**Target:** Widget

**Inputs:**
- In (exec)
- Target (object)
- Direction (enum)
- In Custom Delegate (delegate) — Custom Delegate that will be called

**Outputs:**
- Out (exec)

**Description:** Sets the widget navigation rules for a specific direction. This can only be called on widgets that are in a widget tree. This works only for Custom Rule.

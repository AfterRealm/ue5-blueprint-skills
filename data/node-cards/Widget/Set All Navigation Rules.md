## Set All Navigation Rules
**Category:** Widget
**Search:** "Set All Navigation Rules"
**Target:** Widget

**Inputs:**
- In (exec)
- Target (object)
- Rule (enum) — The rule to use when navigation is taking place
- Widget to Focus (name) — When using the Explicit rule, focus on this widget

**Outputs:**
- Out (exec)

**Description:** Sets the widget navigation rules for all directions. This can only be called on widgets that are in a widget tree.

## Get All Widgets Of Class
**Category:** Widget
**Search:** "Get All Widgets Of Class"
**Target:** Widget Blueprint Library

**Inputs:**
- In (exec)
- Widget Class (class) — The widget class to filter by.
- Top Level Only (boolean) — Only the widgets that are direct children of the viewport will be returned.

**Outputs:**
- Out (exec)
- Found Widgets (object) — The widgets that were found matching the filter.

**Description:** Find all widgets of a certain class.

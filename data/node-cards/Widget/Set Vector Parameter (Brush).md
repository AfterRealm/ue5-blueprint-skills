## Set Vector Parameter (Brush)
**Category:** Widget
**Search:** "Set Vector Parameter (Brush)"
**Target:** MVVMSlate Brush Conversion Library

**Inputs:**
- Brush (struct) — Brush with a material we want to set a parameter on
- Parameter Name (name) — Name of material parameter to set
- Value (linearcolor) — Value to set material parameter to

**Outputs:**
- Return Value (struct)

**Description:** Sets a vector value on a brush material assuming it exists, handles MID existance appropriately

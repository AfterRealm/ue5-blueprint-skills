## Set Scalar Parameter MID (Brush)
**Category:** Widget
**Search:** "Set Scalar Parameter MID (Brush)"
**Target:** MVVMSlate Brush Conversion Library

**Inputs:**
- Brush (struct) — Brush with a material we want to set a parameter on
- Material (object) — Material to set on the brush
- Parameter Name (name) — Name of material parameter to set
- Value (real) — Value to set material parameter to

**Outputs:**
- Return Value (struct)

**Description:** Sets a scalar value on a brush material assuming it exists, handles MID existance appropriately

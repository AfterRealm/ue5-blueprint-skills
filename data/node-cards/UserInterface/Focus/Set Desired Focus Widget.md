## Set Desired Focus Widget
**Category:** UserInterface > Focus
**Search:** "Set Desired Focus Widget"
**Target:** User Widget

**Inputs:**
- In (exec)
- Target (object)
- Widget (object) — Widget to forward the focus to when this widget receives focus

**Outputs:**
- Out (exec)
- Return Value (boolean) — True if the Widget is set properly. Will return false if it's not a child of this UserWidget.

**Description:** Sets the child Widget that should receive focus when this UserWidget gets focus.

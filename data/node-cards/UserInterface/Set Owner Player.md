## Set Owner Player
**Category:** UserInterface
**Search:** "Set Owner Player"
**Target:** Widget Component

**Inputs:**
- In (exec)
- Target (object)
- Local Player (object)

**Outputs:**
- Out (exec)

**Description:** Sets the local player that owns this widget component. Setting the owning player controls which player's viewport the widget appears on in a split screen scenario. Additionally it forwards the owning player to the actual UserWidget that is spawned.

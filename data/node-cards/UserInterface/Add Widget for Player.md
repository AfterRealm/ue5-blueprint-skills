## Add Widget for Player
**Category:** UserInterface
**Search:** "Add Widget for Player"
**Target:** Game Viewport Subsystem

**Inputs:**
- In (exec)
- Target (object)
- Widget (object)
- Player (object)
- Slot (struct)

**Outputs:**
- Out (exec)
- Return Value (boolean)

**Description:** Adds the widget to the game's viewport in the section dedicated to the player. This is valuable in a split screen game where you need to only show a widget over a player's portion of the viewport.

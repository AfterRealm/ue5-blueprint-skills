## Ignore Component when Moving
**Category:** Collision
**Search:** "Ignore Component when Moving"
**Target:** Primitive Component

**Inputs:**
- In (exec)
- Target (object)
- Component (object)
- Should Ignore (boolean)

**Outputs:**
- Out (exec)

**Description:** Tells this component whether to ignore collision with another component when this component is moved. The other components may also need to be told to do the same when they move. Does not affect movement of this component when simulating physics.

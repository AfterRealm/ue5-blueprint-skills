## Get Overlapping Actors
**Category:** Collision
**Search:** "Get Overlapping Actors"
**Target:** Actor

**Inputs:**
- Target (object)
- Class Filter (class) — [optional] If set, only returns actors of this class or subclasses

**Outputs:**
- Overlapping Actors (object) — [out] Returned list of overlapping actors

**Description:** Returns list of actors this actor is overlapping (any component overlapping any component). Does not return itself.

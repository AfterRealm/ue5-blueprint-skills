## Can Character Step Up
**Category:** Collision
**Search:** "Can Character Step Up"
**Target:** Primitive Component

**Inputs:**
- Target (object)
- Pawn (object) — the Pawn that wants to step onto this component.

**Outputs:**
- Return Value (boolean)

**Description:** Return true if the given Pawn can step up onto this component. This controls whether they can try to step up on it when they bump in to it, not whether they can walk on it after landing on it.

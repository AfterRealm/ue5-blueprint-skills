## On Target Perception Forgotten
**Category:** EventDispatchers
**Search:** "On Target Perception Forgotten"

**Description:** Notifies all bound delegates that the perception info has been forgotten for a given target. The notification gets broadcast when all stimuli of a given target expire or the target was explicitly forgotten. The target forgetting functionality must be enabled through the AIPerceptionSystem.bForgetStaleActors flag from Project Settings. @param SourceActor Actor associated to the stimulus (can not be null) @param Stimulus Updated stimulus @see ForgetActor, ForgetAll

## Set Force Mip Levels to be Resident
**Category:** Rendering
**Search:** "Set Force Mip Levels to be Resident"
**Target:** Streamable Render Asset

**Inputs:**
- In (exec)
- Target (object)
- Seconds (real) — Duration in seconds
- Cinematic LODGroup Mask (integer)

**Outputs:**
- Out (exec)

**Description:** Tells the streaming system that it should force all mip-levels to be resident for a number of seconds.

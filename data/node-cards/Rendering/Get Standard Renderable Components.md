**Category:** Rendering
**Search:** "Get Standard Renderable Components"
**Target:** Water Body Component

**Inputs:**
- Target (object)

**Outputs:**
- Return Value (object)

**Description:** Retrieves the list of primitive components that this water body uses when not being rendered by the water mesh (e.g. the static mesh component used when WaterMeshOverride is specified)

## Get Convenient Windowed Resolutions
**Category:** Rendering
**Search:** "Get Convenient Windowed Resolutions"
**Target:** Kismet System Library

**Inputs:**
- In (exec)

**Outputs:**
- Out (exec)
- Resolutions (struct)
- Return Value (boolean) — true if successfully queried the device for available resolutions.

**Description:** Gets the list of windowed resolutions which are convenient for the current primary display size.

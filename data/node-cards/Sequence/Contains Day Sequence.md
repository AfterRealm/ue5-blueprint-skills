## Contains Day Sequence
**Category:** Sequence
**Search:** "Contains Day Sequence"
**Target:** Day Sequence Actor

**Inputs:**
- In (exec)
- Target (object)
- In Day Sequence (object) — the sequence to search for

**Outputs:**
- Out (exec)
- Return Value (boolean) — true if InDaySequence exists in the DaySequences map.

**Description:** Returns true if the given InDaySequence is referenced by any entry in the DaySequences map property.

## Remove Tags from Input Mode
**Category:** Input
**Search:** "Remove Tags from Input Mode"
**Target:** Enhanced Input Subsystem Interface

**Inputs:**
- In (exec)
- Target (interface)
- Tags to Remove (struct) — Tags to remove from the current input mode tag container
- Options (struct) — Options for how to treat the control mappings rebuild

**Outputs:**
- Out (exec)

**Description:** Removes tags from the current input mode and triggers a rebuild of control mappings.

## Get Player Mappable Key Settings
**Category:** Input
**Search:** "Get Player Mappable Key Settings"
**Target:** Enhanced Input Library

**Inputs:**
- Action Key Mapping (struct)

**Outputs:**
- Player Mappable Key Settings (object)

**Description:** Returns the Player Mappable Key Settings owned by the Action Key Mapping or by the referenced Input Action, or nothing based of the Setting Behavior.

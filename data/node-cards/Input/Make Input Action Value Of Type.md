## Make Input Action Value Of Type
**Category:** Input
**Search:** "Make Input Action Value Of Type"
**Target:** Enhanced Input Library

**Inputs:**
- X (real)
- Y (real)
- Z (real)
- Value Type (enum)

**Outputs:**
- Return Value (struct)

**Description:** Builds an ActionValue from X, Y, Z. Inherits type from an existing ActionValue. Ignores axis values unused by the provided value type. Note: Intended for use in Input Modifier Modify Raw overloads to modify an existing Input Action Value.

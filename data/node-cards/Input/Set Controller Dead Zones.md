## Set Controller Dead Zones
**Category:** Input
**Search:** "Set Controller Dead Zones"
**Target:** Player Controller

**Inputs:**
- In (exec)
- Target (object)
- Left Dead Zone (real) — Inner DeadZone for the left analog stick
- Right Dead Zone (real) — Inner DeadZone for the right analog stick

**Outputs:**
- Out (exec)

**Description:** Sets the deadzones of the player's controller

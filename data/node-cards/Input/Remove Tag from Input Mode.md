## Remove Tag from Input Mode
**Category:** Input
**Search:** "Remove Tag from Input Mode"
**Target:** Enhanced Input Subsystem Interface

**Inputs:**
- In (exec)
- Target (interface)
- Tag to Remove (struct) — The tag to remove from the current input mode
- Options (struct) — Options for how to treat the control mappings rebuild

**Outputs:**
- Out (exec)

**Description:** Removes the given tag from the current input mode and triggers a rebuild of control mappings.

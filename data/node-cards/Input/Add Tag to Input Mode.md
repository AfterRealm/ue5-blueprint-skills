## Add Tag to Input Mode
**Category:** Input
**Search:** "Add Tag to Input Mode"
**Target:** Enhanced Input Subsystem Interface

**Inputs:**
- In (exec)
- Target (interface)
- Tag to Add (struct) — The gameplay tag to add to the current input mode tag container
- Options (struct) — Options for how to treat the control mappings rebuild

**Outputs:**
- Out (exec)

**Description:** Adds the given "TagToAdd" to the current input mode tag container and triggers a rebuild of control mappings.

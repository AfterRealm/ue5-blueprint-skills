## Append Tags to Input Mode
**Category:** Input
**Search:** "Append Tags to Input Mode"
**Target:** Enhanced Input Subsystem Interface

**Inputs:**
- In (exec)
- Target (interface)
- Tags to Add (struct) — The gameplay tag container to append
- Options (struct) — Options for how to treat the control mappings rebuild

**Outputs:**
- Out (exec)

**Description:** Appends the given "TagsToAdd" to the current input mode tag container and triggers a rebuild of control mappings.

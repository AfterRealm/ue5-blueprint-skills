## Set Input Mode
**Category:** Input
**Search:** "Set Input Mode"
**Target:** Enhanced Input Subsystem Interface

**Inputs:**
- In (exec)
- Target (interface)
- New Mode (struct) — The new input mode tag container to use
- Options (struct) — Options for how to treat the control mappings rebuild.

**Outputs:**
- Out (exec)

**Description:** Sets the current input mode on the player and triggers a rebuild of control mappings.

## Get Input Mode
**Category:** Input
**Search:** "Get Input Mode"
**Target:** Enhanced Input Subsystem Interface

**Inputs:**
- Target (interface)

**Outputs:**
- Return Value (struct)

**Description:** Returns the current input mode set on Enhanced Input. This mode will control what mappings are currently being processed by the player. If the Input Mapping Context's InputModeQuery is not empty and does not match this tag container, then it's mappings will not be processed.

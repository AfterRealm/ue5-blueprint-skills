## Disable Input
**Category:** Input
**Search:** "Disable Input"
**Target:** Actor

**Inputs:**
- In (exec)
- Target (object)
- Player Controller (object) — The PlayerController whose input events we no longer want to receive. If null, this actor will stop receiving input from all PlayerControllers.

**Outputs:**
- Out (exec)

**Description:** Removes this actor from the stack of input being handled by a PlayerController.

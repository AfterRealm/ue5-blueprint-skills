## Make Oriented Box
**Category:** Math > Oriented Box
**Search:** "Make Oriented Box"
**Target:** Kismet Math Library

**Inputs:**
- Origin (vector) — The center of the box
- Orientation (rotator) — The rotation of the box
- Extent (vector) — The half-size of the box

**Outputs:**
- Return Value (struct) — A new oriented bounding box

**Description:** Makes an FOrientedBox from Center, Orientation and Extent

## Send Key Char
**Category:** Interaction
**Search:** "Send Key Char"
**Target:** Widget Interaction Component

**Inputs:**
- In (exec)
- Target (object)
- Characters (string)
- Repeat (boolean)

**Outputs:**
- Out (exec)
- Return Value (boolean)

**Description:** Transmits a list of characters to a widget by simulating a OnKeyChar event for each key listed in the string.

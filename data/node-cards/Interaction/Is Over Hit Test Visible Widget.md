## Is Over Hit Test Visible Widget
**Category:** Interaction
**Search:** "Is Over Hit Test Visible Widget"
**Target:** Widget Interaction Component

**Inputs:**
- Target (object)

**Outputs:**
- Return Value (boolean)

**Description:** Returns true if a widget under the hit result is has a visibility that makes it hit test visible. e.g. Slate widgets that return true for GetVisibility().IsHitTestVisible().

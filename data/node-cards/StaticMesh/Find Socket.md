## Find Socket
**Category:** StaticMesh
**Search:** "Find Socket"
**Target:** Static Mesh

**Inputs:**
- Target (object)
- In Socket Name (name)

**Outputs:**
- Return Value (object)

**Description:** Find a socket object in this StaticMesh by name. Entering NAME_None will return NULL. If there are multiple sockets with the same name, will return the first one.

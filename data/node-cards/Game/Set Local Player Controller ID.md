## Set Local Player Controller ID
**Category:** Game
**Search:** "Set Local Player Controller ID"
**Target:** Gameplay Statics

**Inputs:**
- In (exec)
- Player (object) — The player controller of the player to change the controller ID of
- Controller Id (integer) — The controller ID to assign to this player

**Outputs:**
- Out (exec)

**Description:** Sets what physical controller ID a player should be using. This only works for local player controllers.

## Get Player Camera Manager
**Category:** Game
**Search:** "Get Player Camera Manager"
**Target:** Gameplay Statics

**Inputs:**
- Player Index (integer) — Index in the player controller list, starting first with local players and then available remote ones

**Outputs:**
- Return Value (object)

**Description:** Returns the camera manager for the Player Controller at the specified player index. This will not include remote clients with no player controller.

## Quit Game
**Category:** Game
**Search:** "Quit Game"
**Target:** Kismet System Library

**Inputs:**
- In (exec)
- Specific Player (object) — The specific player to quit the game. If not specified, player 0 will quit.
- Quit Preference (enum) — Form of quitting.
- Ignore Platform Restrictions (boolean) — Ignores and best-practices based on platform (e.g on some consoles, games should never quit). Non-shipping only

**Outputs:**
- Out (exec)

**Description:** Exit the current game

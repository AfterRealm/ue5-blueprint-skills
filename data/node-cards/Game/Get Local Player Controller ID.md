## Get Local Player Controller ID
**Category:** Game
**Search:** "Get Local Player Controller ID"
**Target:** Gameplay Statics

**Inputs:**
- Player (object) — The player controller of the player to get the ID of

**Outputs:**
- Return Value (integer) — The ID of the passed in player. -1 if there is no physical controller assigned to the passed in player

**Description:** Gets what physical controller ID a player is using. This only works for local player controllers.

## Get Player State from Unique Net Id
**Category:** Game
**Search:** "Get Player State from Unique Net Id"
**Target:** Gameplay Statics

**Inputs:**
- Unique Id (struct) — The player's unique net/online id

**Outputs:**
- Return Value (object)

**Description:** Returns the player state that matches the passed in online id, or null for an invalid one. This will work on both the client and server for local and remote players.

## Set Allows Geometry Selection
**Category:** DynamicMeshComponent
**Search:** "Set Allows Geometry Selection"
**Target:** Dynamic Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- In Allows Geometry Selection (boolean) — Whether geometry elements (triangles, edges, etc) of the dynamic mesh will be interactively selectable.

**Outputs:**
- Out (exec)

**Description:** Enable/Disable interactive geometry element selection for the mesh. Useful to disable geometry selection on procedural meshes where the selection will be frequently invalidated.

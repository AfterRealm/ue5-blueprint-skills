## Update Collision
**Category:** DynamicMeshComponent
**Search:** "Update Collision"
**Target:** Dynamic Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- Only if Pending (boolean) — only update if a collision update is pending, ie the underlying DynamicMesh changed and bDeferCollisionUpdates is enabled

**Outputs:**
- Out (exec)

**Description:** Force an update of the Collision/Physics data for this Component.

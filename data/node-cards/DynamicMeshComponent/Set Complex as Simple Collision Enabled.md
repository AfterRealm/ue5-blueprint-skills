## Set Complex as Simple Collision Enabled
**Category:** DynamicMeshComponent
**Search:** "Set Complex as Simple Collision Enabled"
**Target:** Dynamic Mesh Component

**Inputs:**
- In (exec)
- Target (object)
- Enabled (boolean)
- Immediate Update (boolean) — if true, UpdateCollision(true) is called

**Outputs:**
- Out (exec)

**Description:** If bEnabled=true, sets bEnableComplexCollision=true and CollisionType=CTF_UseComplexAsSimple If bEnabled=true, sets bEnableComplexCollision=false and CollisionType=CTF_UseDefault

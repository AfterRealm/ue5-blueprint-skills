## Connect Material Expressions
**Category:** MaterialEditing
**Search:** "Connect Material Expressions"
**Target:** Material Editing Library

**Inputs:**
- In (exec)
- From Expression (object) — Expression to make connection from
- From Output Name (string) — Name of output of FromExpression to make connection from. Leave empty to use first output.
- To Expression (object) — Expression to make connection to
- To Input Name (string) — Name of input of ToExpression to make connection to. Leave empty to use first input.

**Outputs:**
- Out (exec)
- Return Value (boolean)

**Description:** Create connection between two material expressions

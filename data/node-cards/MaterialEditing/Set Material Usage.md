## Set Material Usage
**Category:** MaterialEditing
**Search:** "Set Material Usage"
**Target:** Material Editing Library

**Inputs:**
- In (exec)
- Material (object) — Material to change usage for
- Usage (enum) — New usage type to enable for this material

**Outputs:**
- Out (exec)
- Needs Recompile (boolean) — Returned to indicate if material needs recompiling after this change
- Return Value (boolean)

**Description:** Enable a particular usage for the supplied material (e.g. SkeletalMesh, ParticleSprite etc)

## Has Material Usage
**Category:** MaterialEditing
**Search:** "Has Material Usage"
**Target:** Material Editing Library

**Inputs:**
- Material (object) — Material to check usage for

**Outputs:**
- Return Value (boolean)

**Description:** Check if a particular usage is enabled for the supplied material (e.g. SkeletalMesh, ParticleSprite etc)

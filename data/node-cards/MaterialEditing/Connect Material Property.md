## Connect Material Property
**Category:** MaterialEditing
**Search:** "Connect Material Property"
**Target:** Material Editing Library

**Inputs:**
- In (exec)
- From Expression (object) — Expression to make connection from
- From Output Name (string) — Name of output of FromExpression to make connection from
- Property (enum) — Property input on material to make connection to

**Outputs:**
- Out (exec)
- Return Value (boolean)

**Description:** Connect a material expression output to one of the material property inputs (e.g. diffuse color, opacity etc)

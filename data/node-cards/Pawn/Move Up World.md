## Move Up World
**Category:** Pawn
**Search:** "Move Up World"
**Target:** Default Pawn

**Inputs:**
- In (exec)
- Target (object)
- Val (real) — Amount of movement in the world up direction (or down if negative).

**Outputs:**
- Out (exec)

**Description:** Input callback to move up in world space (or down if Val is negative).

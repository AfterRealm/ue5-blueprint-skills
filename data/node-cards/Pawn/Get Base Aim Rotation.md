## Get Base Aim Rotation
**Category:** Pawn
**Search:** "Get Base Aim Rotation"
**Target:** Pawn

**Inputs:**
- Target (object)

**Outputs:**
- Return Value (rotator)

**Description:** Return the aim rotation for the Pawn. If we have a controller, by default we aim at the player's 'eyes' direction that is by default the Pawn rotation for AI, and camera (crosshair) rotation for human players.
